R Language--project package management

Jan 11, 2019 | Hits | 306 Words | 1 Minutes

A helpful function to install many package from CRNA or bioLite simultaneously.

[CRAN]
[Bioconductor]

When we handle a complex project, we often need to use many different methods to explore the characteristics of data from different
perspectives. It is economical to use many current methods in R to do that. However, it will spend our many times to install
many useful packages seperately. Besides, we also often encounter some problems like “ package ‘XXX’ is not available “ when using the
command “install.packages(“XXX”). In this situation, we either try to find other source like Bioconductor (use the command: “if (!requireNamespace
(“BiocManager”, quietly = TRUE))install.packages(“BiocManager”)BiocManager::install(“XXX”, version = “3.8”)”), or download the original source
code to recompile it. Here I write a simpler function to help download available packages in CRAN and Bioconductor without intalling or
testing many times.
