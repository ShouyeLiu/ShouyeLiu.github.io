Ubuntu small questions list

Mar 20, 2019 | Hits | 102 Words | 1 Minutes

Question 1, set root password command.

For root :
sudo passwd
For user
sudo passwd username

#sudo apt-get install python-pip is failing

problem:

sudo apt-get install python-pip
Reading package lists… Done
Building dependency tree
Reading state information… Done
E: Unable to locate package python-pip

solve:

sudo apt-get install software-properties-common
sudo apt-add-repository universe
sudo apt-get update
sudo apt-get install python-pip

#git clone https://github.com/shadowsocks/shadowsocks-libev.git
Cloning into ‘shadowsocks-libev’…
fatal: unable to access ‘https://github.com/shadowsocks/shadowsocks-libev.git/': Could not resolve proxy: proxy.hogehoge.co.jp

solve:
git config –global –add remote.origin.proxy “”
