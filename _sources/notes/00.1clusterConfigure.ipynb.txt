    "## Configure conda miniforge environments\n",
    "\n",
    "#### Download [miniforge](https://github.com/conda-forge/miniforge)\n",
    "```bash\n",
    "wget -O Miniforge3.sh \"https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-$(uname)-$(uname -m).sh\"\n",
    "```\n",
    "Here  we need to set conda-forge as the only channel to avoid potential conflicts.\n",
    "\n",
    "```bash\n",
    "conda config --remove-key channels 2>/dev/null || true\n",
    "conda config --show channels\n",
    "conda config --add channels conda-forge\n",
    "conda config --show channels\n",
    "```\n",
    "(condaInstallRpackage)=\n",
    "#### Install r and necessary package into python base env\n",
    "```\n",
    "mamba install -c conda-forge r-base r-tidyverse r-data.table \\\n",
    "    r-bh r-rcpp r-devtools r-remotes\n",
